## 개요

- 현대오토에버 CoE 개발/운영 및 Governance 체계 수립 - Frontend Engineer 전형 지원을 위한 과제 입니다.
- [위블 비즈 FAQ](https://wiblebiz.kia.com/FAQ) 페이지의 클론 코딩입니다.

## 기술 스택

아래의 항목들이 사용되었습니다.

- Next.js
- typescript
- TailwindCSS
- tanstack-query

## 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 에 접속하여 확인할 수 있습니다.
